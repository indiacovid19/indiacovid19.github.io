JSON Data

Click on the following link to view/download the raw JSON data from which this page is generated:

indiacovid19.json

This JSON file contains a list of list entries. Each inner list entry is a septuple (7-tuple) consisting of 7 values. Here is an example entry: 

["2020-03-25", 553, 43, 10, "2020-03-25 18:45",
 "https://web.archive.org/web/20200325192939/https://www.mohfw.gov.in/", ""]

Here is a description of these 7 values in the order they appear in each septuple:

  1. Date (in "YYYY-MM-DD" format as per Indian Standard Time)
  2. Number of active cases
  3. Number of cured cases
  4. Number of death cases
  5. Date and time at which the data was collected/published by the source of the data (in "YYYY-MM-DD HH:MM" format as per Indian Standard Time)
  6. Reference archive link of the source of the data as supporting evidence
  7. Any additional comment or note about the entry

If we add the four numbers in an entry for a specific date, we get the total number of cases for that date.

In this page, the sum of the number of cured cases and that of the migrated cases is reported as the number of cured cases because the MoHFW website too clubs them together in the statewise status tables.

If you find discrepancies in the data, please create new issues to report them or send pull requests to the corresponding GitHub project at GitHub project.

Questions and Answers

  1. Why was this project created when other similar projects already exist?

    Different projects have different conventions for reporting the numbers. Some projects follow the latest updates appearing in media to estimate the case numbers in real time. As a result, the case numbers in such projects often differ from the numbers officially published by Government of India. This project does not provide case numbers in real time. Instead, this project archives the case numbers for each date from official sources only.

  2. What sources are used for the case numbers archived in this project?

    The case numbers published as part of press releases on pib.gov.in and the case numbers published on mohfw.gov.in are used as the sources of the case numbers archived in this project.

  3. Where can I find the original sources of the case numbers?

    For each row in the case numbers table, the last column contains reference archive links that point to archived snapshots of press releases or pages from pib.gov.in and mohfw.gov.in that contain the original data related to the case numbers.

  4. Where can I ask more questions or provide feedback about this project?

    To ask questions or provide feedback either post new issues to this project or send a tweet to the author.

  5. How is this website generated?

    This website is generated using a couple of Python scripts available at github.com/indiacovid19. The static site generator used for this project is a modified version of Sunaina Pai's makesite.py. Here are the commands to generate this website: 

    git clone https://github.com/indiacovid19/indiacovid19.git
cd indiacovid19
make venv
make site

  6. What is the script wiki.py in the source code meant for?

    The script wiki.py generates Wikipedia markup to plot the graphs used in the Wikipedia article on 2020 Coronavirus Pandemic in India. Here are the commands to generate the markup: 

    git clone https://github.com/indiacovid19/indiacovid19.git
cd indiacovid19
make wiki

    The above commands generate the following output files:

    • wiki1.txt: Wikipedia markup for India medical cases chart template.
    • wiki2.txt: Wikipedia markup for India medical cases template.
    • wiki3.txt: Wikipedia markup for India statistics charts.

    Copies of both files are available on this website too. Click on the filenames in the list above to view the corresponding markups.
